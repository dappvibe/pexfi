# A New Way to Trade Crypto

PEXFI is a fully decentralized, non-custodial peer-to-peer marketplace for trading crypto and fiat. No KYC, no intermediaries, no frozen accounts.

## Key Features

- **Smart Contract Escrow** — Open-source contracts lock crypto until trade completes
- **Reputation NFT** — Non-transferable token tracks your trading history
- **On-Chain Messaging** — Communicate with counterparties directly on blockchain
- **Multiple Payment Methods** — Support for various fiat payment options
